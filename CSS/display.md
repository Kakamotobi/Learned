# Display
> Sets an element's inner and outer display types.

## Inner Type
- Sets the layout of children, such as flow layout, `grid` or `flex`.

## Outer Type
- Sets an element's participation in flow layout.

### `block`
- Break the normal flow of the document.
- Take up the entire width of their parent element by default.

### `inline`
- `width` and `height` do not work on `inline` elements.
- Always stay the size of the content inside them.
- Inline elements gets pushed down to different lines when they don't fit in their parent element's width.

### `inline-block`
- Puts elements on the same line while also allowing changes to their size.
