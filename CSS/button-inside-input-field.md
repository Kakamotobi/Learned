# Button Inside Input Field
```
<HTML>
<body>
  <div class="container">
    <input type="text">
    <button>Button</button>
  </div>
</body>

<CSS>      
.container {
  position: relative;
}
      
button {
  position: absolute;
  top: 0;
  right: 0;
  cursor: pointer;
}
```
